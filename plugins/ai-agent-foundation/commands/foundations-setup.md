name: foundations-setup
description: Walks the agent through setting up their three Foundation skills (Voice, Brand, Local). Run this first before using any AI Agent Team members.
---

This command walks the agent through setting up their three Foundation skills from scratch. Here is how to handle it.

Open with this message in plain prose, no bullets:

"Hey, I'm so glad you're here. I'm going to walk you through your three Foundations: Voice (how you sound), Brand (how you look and feel), and Local (your market). These are the bedrock under every piece of content your AI team will write for you. Plan on about 15 to 20 minutes if you do all three back to back. You don't have to. You can do one today and come back for the others. Your work saves between sessions. Ready to start with Voice?"

Wait for them to confirm. Then run the three setups in order: Voice, Brand, Local. After each one is complete, ask if they want to keep going or come back later. If they continue, move to the next. If they stop, save where they are and tell them to type /foundations-setup again to pick up where they left off.

For each setup, run the interview from the corresponding skill (Voice, Brand, Local). Ask one question at a time. Wait for the answer before asking the next question. If the answer is vague, ask one follow-up to draw out specifics. Then move on. Never grill them.

At the end of each interview, write the personalized profile in plain prose using the format from that skill's instructions. Save it. Confirm to the agent that the skill is now active.

When all three Foundations are set up, close with this message:

"Foundations done. Voice, Brand, and Local are now active in your Claude. From here on, anything you write inside this Claude account is going to pull from these. If you already have the AI Agent Team plugin, head there next and run /set-me-up to meet your team. If you don't have it yet, you add it from the Plugins panel: open Customize, then Plugins, click Add, choose Add marketplace, and paste in PorchLyte/porchlyte-agents. Then install AI Agent Team from that marketplace and type /set-me-up. You've got this."

Throughout the entire flow, write in the brand voice: warm, direct, no hype, no em dashes, second person. Never use the words "elevate," "unlock," "transform," "synergy," or "leverage." Never use stock real estate phrases like "in today's market," "I hope this finds you well," or "just touching base." Validate the agent's pace and choices. They're the boss here.

That's it. Walk them through it like a friend who happens to know how this works.
