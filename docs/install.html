SET UP YOUR
FOUNDATION.

Before you meet your AI team, you teach Claude how you sound, how your brand looks, and what your market is really like. That's the Foundation. Every agent you hire later reads from it.

Set this up first. Your team comes on a later day, so you're never doing everything at once.

The Foundation Setup

0 of 6 complete
01
Get Claude Pro ($20/month).
Go to claude.ai, create an account if you don't have one, and upgrade to Claude Pro. The free version won't run the plugins. Pro is the plan you need.
02
Download the Claude desktop app.
The plugins live inside a feature called Cowork, which only runs on the desktop app (Mac or Windows). Not the browser. Not mobile. Download it from claude.ai once you've signed in to Pro.
03
Turn off "Help improve Claude."
This one matters. In the desktop app, go to Settings > Privacy > Help improve Claude and toggle it off.
When this setting is on, Anthropic can use your conversations to help train future versions of Claude. When it's off, your conversations stay yours. You're going to be talking about client situations, listing details, and personal market data. Keeping that out of any training pipeline is a smart layer of privacy, especially as a real estate agent.
Tip
Do this before you run any of the setup commands below, so nothing flows through with the setting still on.
04
Add the Foundation marketplace.
Open a new chat in the Claude desktop app. Type this command and press Enter. You only do this once.
/plugin marketplace add PorchLyte/porchlyte-foundations
Tip
Run each command in its own message. Wait for Claude to finish before you send the next one.
05
Install AI Agent Foundation.
In the same chat, type the install command. This is the only thing you install today.
/plugin install ai-agent-foundation@porchlyte-foundations
Tip
To confirm it's active, open Customize > Plugins. You should see AI Agent Foundation listed.
06
Run /foundations-setup.
Open a new chat and type the command below. Claude walks you through a short interview about your voice, brand, and local market, one at a time. Plan on about 15 to 20 minutes for all three. You can stop after one and come back later. Your answers save.
/foundations-setup

Getting updates

When the Foundation is improved, you don't reinstall anything.

/plugin marketplace update porchlyte-foundations

Run that in any Cowork chat and Claude pulls the latest version. Your Voice, Brand, and Local answers stay exactly as you set them up. Updates only change how the Foundation works, not your personalization.

FAQ

Can I use the free version of Claude?+
No. The plugins require Cowork, which comes with Claude Pro. $20 per month, billed by Anthropic directly.
Do I have to do the Foundation before the Team?+
Yes. The Foundation is the personalization layer every team member reads from. Without it, the Team works but the writing comes out generic instead of sounding like you. Foundation first, Team on a later day.
Can I use this on my iPad or phone?+
Not right now. Cowork is desktop only (Mac or Windows). You can use Claude on mobile for other things, but the setup commands won't be available there.
How long does the Foundation take?+
The install takes seconds. The /foundations-setup interview is the main part: about 15 to 20 minutes for all three (Voice, Brand, Local). You can do one today and come back for the others. Your work saves.
Can I change my answers later?+
Yes. Re-run /foundations-setup any time and update anything that's changed. Your voice and market evolve, and the Foundation evolves with you.
What if a slash command doesn't work?+
First, make sure the plugin is installed (Customize > Plugins). Second, start a brand new chat before running /foundations-setup. Third, close and reopen the desktop app. That solves it about 95% of the time.
Is my client data safe?+
With the "Help improve Claude" setting turned off (step 03), your conversations are not used for training. Treat your Foundation the way you'd treat any tool that holds business information: don't share more than you need to, and follow your brokerage's data handling rules.
